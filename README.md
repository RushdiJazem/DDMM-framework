# DDMM: A Structurally Closed Framework for Internalizing Relations in Information Systems

This paper introduces the Dynamic Distributed Memory Model (DDMM), a unified structural framework in which both entities and relations are represented using a single primitive type (cells). The model supports the internal construction of higher-order structures through composition while remaining within a single representational space.
